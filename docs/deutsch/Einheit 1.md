## 人称代词第一格
|      | 第一人称 | 第二人称 | 第三人称  | 尊称 |
| ---- | -------- | -------- | --------- | ---- |
| 单数 | ich      | du       | er/sie/es | Sie  |
| 复数 | wir      | ihr      | sie       | Sie  |

### 示例
* Das Bild (Es) ist schön.
  * 画（她）很好看。
* Der Tisch (Er) ist groß.
  * 桌子（他）很大。
